coreboot™

coreboot™ is open source firmware for custom solutions. It can be found in devices from low power embedded systems to super computer clusters. Happy 15th birthday to coreboot.

"coreboot™: fast and flexible Open Source firmware. coreboot™ is an extended firmware platform that delivers a lightning fast and secure boot experience on modern computers and embedded systems. As an Open Source project it provides auditability and maximum control over technology."

- coreboot.org
